# Баг-репорт. Задание 1: Скриншот с багами

## Тестовое задание для стажёра QA-направления (весенняя волна 2025)

- В результатах поиска присутствует объявление, не соответствующее категории (плюшевая игрушка вместо живого котёнка) — P1
- Фильтр по географии не работает: отображается объявление из Новосибирска при поиске по Москве — P1
- Фильтр по цене не работает: в выдаче присутствуют объявления с ценой 500 и 40 000 руб., выходящими за пределы указанного диапазона (1 000–30 000 руб.) — P1
- Фильтр по полу не работает: в выдаче присутствует объявление с котёнком женского пола при выбранном фильтре «Мальчик» — P1
- Ошибка загрузки страницы: отображается сообщение «Ошибка, обновите страницу» — P1
- В разделе «Подобрали для вас» отображаются бенгальские кошки вместо персидских — P2
- В результатах поиска присутствует объявление породы «Экзотическая короткошёрстная» при активном фильтре по породе «Персидская» — P2
- В фильтре «Вид объявления» присутствует нерелевантный пункт «Вязка» при поиске котят — P2
- Фильтр по возрасту отображает некорректное значение «Щенок» в поиске по котятам — P2
- Фильтр по возрасту избыточен: при поиске по запросу «Котята» возраст уже определён, отдельный фильтр не требуется — P2
- При наведении на кнопку «Добавить в избранное» всплывает подсказка от другой кнопки: «Не показывайте мне похожие объявления» — P2
- В фильтре геолокации доступен переключатель «Сначала из Можайска», не соответствующий зоне поиска по Москве — P2
- В хлебных крошках некорректный порядок категорий: «Персидская → Кошки» вместо «Кошки → Персидская» — P3
- Опечатка в пункте меню: «Коръера» вместо «Карьера» — P3
- Опечатка в тексте кнопки: «Уведомять о новых» вместо «Уведомлять о новых» — P3
